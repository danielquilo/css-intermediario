
# 5 Formas de Trabalhar com a Propriedade Position do CSS

### Aula 01
# Position Relative 
![Position Relative](Position-Relative.png)
#### Elementos com posicao relativa sao posionados em relacao a sua posicao original.Eles podem ser deslocados usando propriedades com top,right,bottom e left.



### Aula 02
# Position Absolute
![Position Absolute](Position-Absolute.png)
### Elementos com a posicao absoluta sao posicionados em relacao ao elemento pai mais proximo que possui uma posicao definida.<strong diferente de static </strong> e podem ser deslocados usando propriedades como top,right,bottom e left.

# Aula 03
# Position Fixed
![Position Fixed](Position-Fixed.png)
### Elementos com posicao fixa sao posicionados em relacao a janela do navegador e permanecem fixoz, mesmo quando a pagina e rolada. Um elemento fixo nao deixa lacuna na pagina onde normalmente estaria localizado.

# Position Sticky
![Position Sticky](Position-Sticky.png)
### Elementos com posicao sticky sao posicionados com base no fluxo do documento. ate que sejam deslocados alem de um ponto especifico, onde entao ficam colocados na posicao definida.



## Position Static 
![Position Static](Position-Static.png)
### Posicionamneto padrao "static".Elementos com posicao estatica seguem o fluxo normal do documento.
#### um elemento "static" segue o fluxo normal do documento e nao e afetado por outras propriedades de posicionamento.
